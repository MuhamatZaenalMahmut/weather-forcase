## Design Inspiration
- https://dribbble.com/shots/9325012-Weather-App

## Intallation

- Git Clone
```
$ git clone https://github.com/MuhamatZaenalMahmut/eventapp
```

- Yarn Install
```
$ yarn install
```

- Running in Android
```
$ react-native run-android
```
